## Where this rib fits

The rib bridges the M365 assistant and data plane that WorkIQ exposes:
Copilot answers over your mail, meetings, and files, plus Microsoft
Graph entity operations. The harness stays domain-free; everything
WorkIQ-shaped lives here. The generic rib contract itself belongs to
keelson's docs; this site documents one concrete rib that implements
it, and does so as plainly as possible because this rib doubles as the
reference for writing your own.
